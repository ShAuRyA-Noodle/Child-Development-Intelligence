# AI Early Childhood Intelligence System

Frontend application for early childhood intelligence workflows, including risk stratification, intervention planning, and governance dashboards.

## Development

Prerequisites:
- Node.js 18+
- npm

Run locally:

```sh
npm install
npm run dev
```

Build for production:

```sh
npm run build
```

## Tech stack

- Vite
- React
- TypeScript
- Tailwind CSS
- shadcn/ui
